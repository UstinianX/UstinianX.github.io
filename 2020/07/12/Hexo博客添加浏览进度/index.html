使用$\text{NexT}$主题内置的插件实现浏览进度的显示

打开 themes/next/_config.yml ,搜索关键字 scrollpercent ,把 false 改为 true 以启动插件

如果想把 top 按钮放在侧边栏,打开 themes/next/_config.yml ,搜索关键字 b2t ,把 false 改为 true，这一行就在 scrollpercent 的上面
